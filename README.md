# 🌱 AI-Based Precision Agriculture System

## 📌 Overview

This project presents an **AI-based Precision Agriculture System** that recommends the most suitable **crop type, fertilizer, and water requirements** based on soil and environmental conditions. The system leverages **Machine Learning (Random Forest)** and data analysis techniques to improve agricultural productivity and resource efficiency.

---

## 🚀 Features

* 🌾 Crop Recommendation based on soil nutrients and weather conditions
* 🌱 Fertilizer Prediction using NPK values
* 💧 Water Requirement Estimation (based on crop type)
* 📊 Exploratory Data Analysis (EDA) with visualizations
* 📈 Feature Importance Analysis using ML models

---

## 📂 Dataset Features

The dataset includes the following parameters:

* Temperature
* Humidity
* Soil Moisture
* Soil Type
* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Rainfall
* Crop Type
* Fertilizer Name

---

## ⚙️ Methodology

### 1. Data Preprocessing

* Handling missing values
* Encoding categorical features (Soil Type, Crop Type, Fertilizer Name)

### 2. Exploratory Data Analysis (EDA)

* Correlation Matrix
* Scatter Plots
* Distribution Analysis

### 3. Model Development

* **Random Forest Classifier** used for:

  * Crop Prediction
  * Fertilizer Recommendation

### 4. Evaluation

* Accuracy Score
* Feature Importance Analysis

## 📊 Results

* Rainfall and humidity identified as most influential features
